# lfsr128sum (v2.1)

Высокопроизводительная C++20 библиотека и консольная утилита для вычисления 128-битных контрольных сумм на базе парных LFSR-генераторов (Linear Feedback Shift Register). Проект сочетает классическую математику регистров сдвига с современными SIMD-инструкциями.

## 🚀 Ключевые особенности

*   **Экстремальная производительность**: Скорость обработки данных достигает **~1.6 ГБ/с** на одно ядро (архитектура x86-64 с поддержкой AVX2/SSE4.2).
*   **SIMD-оптимизация**: Векторная обработка 8 ячеек состояния одновременно с использованием `_mm_alignr_epi8`, `_mm_mullo_epi16` и быстрой редукции Барретта.
*   **Математическая обратимость**: Уникальная поддержка функций `back_simd_block`, позволяющих "отматывать" состояние генератора назад на 128 бит за один шаг.
*   **Гарантированное качество**: 100% Coverage (покрытие) на 64-битных тестах. Высокая диффузия входных данных исключает коллизии на коротких дистанциях.
*   **Интеллектуальная соль (Salt)**: Учет реального размера файла в соли предотвращает коллизии для файлов, различающихся только количеством нулевых байтов в конце.
*   **Современный стек**: C++20 (`std::span`, `std::filesystem`, concepts), CMake и кроссплатформенная поддержка (Windows/MSVC, Linux/GCC).

## 🛠 Сборка проекта

Проект не имеет внешних зависимостей. Для сборки требуется компилятор с поддержкой C++20.

```bash
# Клонирование
git clone https://github.com
cd lfsr128sum

# Сборка в режиме Release (важно для SIMD оптимизаций)
cmake -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build --config Release
```

## 🖥 Использование

### Консольная утилита
Программа работает как классический хэш-сумматор. Поддерживаются режимы вычисления хэша, тестирования и замера скорости.

*   **Вычисление хэша файла**:
    ```bash
    ./lfsr128sum <путь_к_файлу>
    ```
*   **Запуск тестов на покрытие и обратимость**:
    ```bash
    ./lfsr128sum --test
    ```
*   **Запуск встроенного бенчмарка скорости (MiB/s)**:
    ```bash
    ./lfsr128sum --bench
    ```

### Использование как C++ библиотеки
Благодаря шаблонам и интринсикам, библиотеку легко интегрировать в свои проекты для быстрой генерации ПСП или хэширования потоков данных.

```cpp
#include "lfsr_hash.h"
#include <immintrin.h>

// 1. Инициализация генератора
lfsr_hash::gens generator;
generator.reset(); // Установка начального состояния {1, 0...}

// 2. Обработка данных блоками по 16 байт (128 бит) через SIMD
alignas(16) uint8_t data[16] = { /* ваши данные */ };
__m128i block = _mm_loadu_si128(reinterpret_cast<const __m128i*>(data));

// Шаг вперед (математически эквивалентен 1 векторизованному такту)
generator.g_251x4.next_simd_block(block);

// 3. Математическая обратимость (откат состояния точно на 1 шаг назад)
generator.g_251x4.back_simd_block(block);

// 4. Получение финального 128-битного результата
lfsr_hash::u128 final_hash = generator.form_hash128();
